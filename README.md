# My First Alarm App
Welcome to my first alarm app! I created this app to address some of the shortcomings I found in Apple's native alarm app, drawing inspiration from features available on Android.

#Key Features

Set Alarms for Specific Dates: One of the main motivations behind this app was the lack of ability to set an alarm for a specific date in Apple's alarm app. As a forgetful person myself, I often relied on alarms to remind me of important events, and the absence of this feature was a significant drawback. My app fills this gap, allowing you to set alarms for any desired date. 
Intelligent Default Settings: In Apple's alarm app, every new alarm is set to never ring by default unless you explicitly specify the days. I found this counterintuitive and prone to missed alarms. In my app, alarms are automatically set to ring every day by default, ensuring you don't forget to set the repeat settings. 
Intuitive Alarm Management: Managing alarms should be a breeze. In my app, you can easily select the "Everyday" option with a single tap, eliminating the need to manually select each weekday. Additionally, deleting an alarm is as simple as swiping right, providing a more intuitive and familiar gesture. 

# Motivation
After experiencing these frustrations with Apple's alarm app and not finding any suitable alternatives in the App Store, I decided to take matters into my own hands. I thought, "Why not create my own alarm app that addresses these issues and provides a better user experience?" 
Thus began my journey of developing this alarm app. It has been a rewarding experience, allowing me to learn and grow as a developer while creating a tool that I believe will be helpful to myself and others who share similar needs. 
I hope you find this alarm app useful and enjoy using it as much as I enjoyed building it! If you have any feedback or suggestions, please don't hesitate to reach out. 
Happy alarm-setting!
